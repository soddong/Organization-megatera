## 학습 키워드
- Domain Model 이란
- Repository
- VO(Value Object)

## Domain 
* 정의 : 비즈니스 로직을 수행하는 객체
- Domain Model 이란   
	도메인에 대한 지식을 선택적으로 단순화하고 의식적으로 구조화한 형태

## Repository   
* 영구 저장소가 아닌, 객체의 상태를 관리하는 저장소

## VO(Value Object)
* dto 패키지
* 불변 클래스로, Read-Only 속성만을 가지고 있음 (getter 성격의 메서드민 사용)

(업데이트 예정)
